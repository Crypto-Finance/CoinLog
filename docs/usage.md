# Usage Guide

This guide explains how to use CoinLog to track and analyze your cryptocurrency trades.

## 1. Manual Trade Entry

1. Navigate to "Import" → "Manual Entry"
2. Fill in trade details (exchange, symbol, direction, entry/exit price, quantity, fees, PnL)
3. Optional: Add annotations (setup type, R:R, stop loss, take profit, market condition, notes)
4. Click "Add Trade"

## 2. Import from Bybit

1. Navigate to "Import" → "Bybit"
2. Enter your Bybit API key and secret (encrypted before storage)
3. Optional: Filter by symbol and date range
4. Click "Import Trades"

## 3. Import from CSV

1. Navigate to "Import" → "CSV"
2. Download the sample CSV template
3. Fill in your trade data following the template format
4. Upload the CSV file

## 4. View Statistics

1. Navigate to "Stats" in the sidebar
2. View overall stats, monthly breakdown, and performance metrics
3. Stats update automatically when trades are added/modified

## 5. Export Data

1. Navigate to "Settings" or use the export button
2. Click "Export to CSV"
3. Download your complete trade history
